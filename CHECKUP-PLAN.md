## Структура статьи /checkup/

### URL
`/checkup/<klyuchevaya-fraza>/`

### Формат
- Breadcrumbs: Главная → Самопроверка → Название статьи
- H1 = поисковый запрос
- Lead (1 абзац, суть)
- 5–7 секций с h2/h3
- CTA-блок с телефоном
- Блок «Читайте также» со ссылками на смежные страницы

### Требования к контенту
- Тон: профессиональный, для пациентов, без сложной терминологии
- Включить: клинические показания, биосовместимость материалов
- Использовать информацию из PubMed и зарубежных стоматологических источников (dentaly.org, animated-teeth.com, dentistry.com, www.scirp.org/index) — как факты, без ссылок
- Заголовки на русском
- Секции с h2 и h3
- Без упоминания виниров
- CTA только телефон (никаких Telegram, WhatsApp)

## Мониторинг
- После публикации всех статей — повторный замер Wordstat через 2–3 месяца
- В Яндекс.Вебмастере отслеживать: показы, клики, позиции по каждому запросу
